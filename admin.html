1 Install Python Dependencies

On your server, install the required Python packages for the scraper:

pip install requests beautifulsoup4 lxml schedule

2 Upload Scraper Files to Your Server

Upload these files to your web server (e.g. via FTP or cPanel File Manager):

scraper.py ← The main scraper seed_db.py ← Run once to add demo data sweepstakes_db.json ← Approved listings (auto-created) pending_review.json ← Pending queue (auto-created) rejected.json ← Rejected entries (auto-created) site_sweepstakes.json ← What your website reads

3 Schedule Daily Automatic Scraping (Cron Job)

In cPanel → Cron Jobs, add this to run the scraper every morning at 6 AM:

0 6 * * * cd /home/yourusername/public_html && python3 scraper.py >> scraper.log 2>&1

Or use the built-in scheduler:

python3 scraper.py --schedule # Runs continuously, scrapes at 6 AM daily

4 Set Up Google Alerts RSS (Recommended)

Go to google.com/alerts and create alerts for:

"sweepstakes" "no purchase necessary" "enter to win" sweepstakes "sweepstakes" site:prnewswire.com

Set delivery to "RSS feed", copy the feed URL, paste into ALERT_RSS_URLS in scraper.py.

5 Connect Your Website to the Database

Your site pages (index.html, category.html, etc.) should fetch site_sweepstakes.json to display live listings. Add this JavaScript to your pages:

fetch('/site_sweepstakes.json') .then(r => r.json()) .then(data => { data.sweepstakes.forEach(sweep => { // Build and insert listing row HTML renderListing(sweep); }); });

6 WordPress / PHP Option

If you run WordPress, install the WP All Import plugin and point it at site_sweepstakes.json — it will auto-import new listings as posts. Or use a simple PHP file to read the JSON:

<?php $data = json_decode(file_get_contents('site_sweepstakes.json'), true); foreach ($data['sweepstakes'] as $sweep) { echo "<div class='listing'>{$sweep['title']}</div>\n"; } ?>
